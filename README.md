# LST

Productivity tools for planning and focusing on everyday tasks.

![App Screenshot](./public/lst.png)

## Features

- Todos
- Pomodoros
- Timers
- Workflows
- Themes; and
- Locally stored data


## Roadmap

- Offline usuage
- Add more tools
- Mobile app
- Desktop app

## Tech Stack

React - TypeScript - Tailwindcss - Nextjs 

## Run Locally

Clone the project

```bash
  git clone https://github.com/fromenoabasi/lst.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
